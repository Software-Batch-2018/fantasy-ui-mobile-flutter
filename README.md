## Tech Stack

Flutter, GetX for global state management and Material Ui.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Software-Batch-2018/fantasy-ui-mobile-flutter
```

Go to the project directory

```bash
  cd fantasy-ui-mobile-flutter
```


Run Application

```bash
  flutter run
```
